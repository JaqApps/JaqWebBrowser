# JaqWebBrowser
Jaq WebBrowser/NewTab

# What is this?
Jaq is a WebBrowser/NewTab! it is mostly made for people that are using old computers such as Windows XP.

Opera (Main)

FireFox

Chrome (UnSupported (Including Chromebooks))

Chromium (Unsupported)
